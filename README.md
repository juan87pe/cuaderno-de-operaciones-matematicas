# Cuaderno de Operaciones Matemáticas Interactivo 📝✨

Un cuaderno digital interactivo diseñado para ayudar a los niños a practicar operaciones matemáticas básicas (suma, resta, multiplicación y división) de forma visual, estructurada y divertida, simulando la experiencia de un cuaderno escolar de cuadrícula con herramientas interactivas avanzadas.

## 🚀 Características Principales

- **🎨 Interfaz de Cuaderno Escolar:** Diseño premium que simula las hojas cuadriculadas tradicionales utilizando una tipografía limpia y amigable para niños (*Fredoka*).
- **♾️ Cuadrícula Infinita:** El lienzo crece automáticamente en cualquier dirección si el usuario escribe, pinta o navega cerca de los bordes.
- **⌨️ Teclado Virtual Integrado:** Un panel lateral táctil que incluye números, operadores aritméticos básicos (`+`, `-`, `x`, `÷`), funciones especiales como tachar celdas (diagonal de llevadas) y colocar líneas de operación naranja.
- **⚡ Carga Inteligente de Operaciones:** Permite ingresar operaciones directamente (ej. `42450 - 18930` o `84 / 6`) y las coloca automáticamente con la alineación y formato escolar correcto (galera en divisiones, filas alineadas a la derecha para sumas/restas).
- **✅ Verificación y Feedback Pedagógico:**
  - Valida el resultado dígito por dígito respetando el orden posicional (unidades, decenas, centenas).
  - Feedback visual inmediato: Celdas correctas se iluminan en verde menta y celdas incorrectas/desplazadas en rojo suave.
  - Mensajes de orientación para motivar el aprendizaje en lugar de solo penalizar el error.
- **⌨️ Soporte Completo de Atajos de Teclado:**
  - Flechas de navegación (`↑`, `↓`, `←`, `→`) para moverse libremente por la cuadrícula.
  - Tecla `-` o `_` para pintar la línea naranja de operación de forma rápida.
  - Tecla `Enter` y `Backspace` para navegación fluida.
- **🧼 Herramientas de Diseño:** Paleta de colores para pintar celdas individuales (celeste, amarillo con borde verde, rosa) y borrador.

## 🛠️ Tecnologías Utilizadas

- **HTML5** & **JavaScript (ES6+)**
- **Tailwind CSS** (para diseño ágil y responsivo)
- **Google Fonts** (Fredoka)

## 📦 Cómo Ejecutar el Proyecto

Dado que es una aplicación web del lado del cliente, no requiere de servidores de base de datos ni compilaciones complejas. Solo necesitas abrir el archivo en tu navegador:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/juan87pe/cuaderno-de-operaciones-matematicas.git
   ```
2. Abre el archivo `cuaderno_de_matem_ticas_interactivo.html` en cualquier navegador web moderno (Chrome, Edge, Firefox, Safari).
